========================================================
- evalaute, show, formate code in document (or not)
- Sweave: generate PDF

  + can be difficult
- R Markdown: HTML, PDF, Word  
  + easy to format  
  + insert code chunks easily  
  + [additional engines available (e.g., Python, Awk, Perl)](http://yihui.name/knitr/demo/engines/)
  + LaTeX exported
